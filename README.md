# Iris Flower Classification - CodeAlpha Internship
## 📌 Project Overview
This project is part of the CodeAlpha Data Science Internship. The objective is to classify iris flower species using machine learning techniques.
## 🌸 Dataset
The Iris dataset contains measurements of three flower species:
- Setosa
- Versicolor
- Virginica
Features used:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
## 🤖 Model Used
Logistic Regression algorithm is used for multi-class classification.
## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Train-Test Split
- Classification Metrics
## 📊 Evaluation Metrics
- Accuracy Score
- Precision
- Recall
- F1-Score
## 🎯 Result
The trained model achieved high accuracy in predicting iris species.
